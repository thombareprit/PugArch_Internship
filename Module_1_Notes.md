MODULE 1.1

HTML - Hyper Text Markup Language

Basic Structure -
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Page Title</title>
  </head>
  <body>
    <!--content-->
  </body>
</html>

Tags - 
    Start: <tag> end: </tag>

    1. <h1> to <h6> : Headings
    2. <p> : Paragraph
    3. <a herf = "...">title<a> : Hyperlink
    4. <img src="..." alt="..."> : Image
