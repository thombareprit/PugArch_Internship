# MODULE 1.1

## HTML : Hyper Text Markup Language

**Basic Structure:**

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Page Title</title>
  </head>
  <body>
    <!--content-->
  </body>
</html>
```

## TAGS :

Start : `<tag>` <br>
End : `</tag>`

  1. `<h1> to <h6>` : Headings
  2. `<p>` : Paragraph
  3. `<a herf = "...">title<a>` : Hyperlink
  4. `<img src="..." alt="...">` : Image


# MODULE 1.2

## LISTS :

  **1. Unorderd List :**
  
  ```
    <ul>
      <li>...</li>
    </ul>
  ```

  **2. Orderd List :** 
  
  ```
    <ol>
      <li>...</li>
    </ol>
  ```

## JUMP LINKS : Same Page Navigation

  ```html

  <h2 id = "section1"> Section 1 </h2>
  <a href = "#section1"> Jump to Section 1</a>
  ```
